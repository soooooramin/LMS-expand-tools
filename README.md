# LMS-expand-tools
ver. 1.0 <br>
Apr 10, 2026

## 概要
これは学務情報サービスを経由せずにLMSにアクセスするためのツールです。
技術スタックは一通り把握していますが、AI（ClaudeCode）に修正を依頼した部分があるのでセキュリティ的な部分は保証しません。
また、これは現時点（2026/4/9時点）でベータ版です。
使用は自己責任でお願いします。

### 背景
もとのLMSのアクセス方法は、「現在の学務情報サービスを利用してのアクセス」と「WebClassからの直接アクセス」の２通りでした。しかし、多くの人が後者を使用していたためたびたびアカンサスポータルにアクセスしずらい状況となっていました。
（おそらく２つのサーバーがあり、一つは学務情報サービス専用、もう一つはLMS,WebClass用というように利用していたのだとおもいます。）
その負荷軽減のため、2026年2月のシステム改定に、学務情報サービスを経由してLMSにアクセスする必要が出てきました。
（多分学務の方と負荷を分散させたかったのかな？）
しかし、蓋を開けてみればシステム改定後の方がアクセスしづらい状況だし、アクセスしづらいしでイライラしたわけです、僕は。
そこで、以前のように、比較的使いやすいように作った拡張機能がこれです。需要があるようなので配布します。
大学のシステムに関しては他にも色々言いたいことがあるのですが、今回は伏せます。それに関しても機能の追加をしていけたらいいなとおもいます。

## 使い方 (FireFox)
推奨はしませんが、このリポジトリの「Ff-lms-expand-tools」をクローンすることで、FireFoxの公式アドオンストア公開前でも導入することができます。その場合開発者モードを用いて導入する必要がありますが、データの保存はできないのでご注意ください。導入については下記URLが参考になると思います。
[Mozilla「初めての拡張機能」](https://developer.mozilla.org/ja/docs/Mozilla/Add-ons/WebExtensions/Your_first_WebExtension#%E3%82%A4%E3%83%B3%E3%82%B9%E3%83%88%E3%83%BC%E3%83%AB)
### 導入
1. FireFoxのアドオン（ff-lms-expand-tools）を導入してください。
2. アドオンの設定画面から授業名、LMSにアクセスするためのURLを入力してください。LMSにアクセスするためのURLは以下のように取得してください
   ~~~
   学務情報サービス＞履修・成績情報＞履修時間割表＞登録する授業名をクリック＞「LMS」欄のジャンプボタンを右クリックし、「リンクをコピー」をクリックし取得する
   ~~~
3. 「保存する」をクリックし、データを保存。
4. アカンサスポータルを開くと、右下に「LMSを開く」が表示されているので、これをクリック
5. タイムテーブルに先ほど入力した内容が記載されているのを確認し、授業をクリック
6. ページに飛びます

### 時間割の変更
アドオンの設定画面から授業名とURLを変更し、保存してください。
**まだベータ版のため、１つの教科を入力するごとに保存しておくことをお勧めします。**

## 使い方 (Chrome)
推奨はしませんが、このリポジトリの「Ch-lms-expand-tools」をクローンすることで、Chromeの公式Webストア公開前でも導入することができます。これはFireFoxとは違い再起動後もデータは紛失しません。導入については下記URLが参考になると思います。[けーちゃんのプログラム開発ノート「自作Chrome拡張機能をテストする方法」](https://note.affi-sapo-sv.com/chrome-extent-test.php)
### 導入
1. Chromeの拡張機能を導入してください。
2. 拡張機能の設定画面（extention options）から授業名、LMSにアクセスするためのURLを入力してください。LMSにアクセスするためのURLは以下のように取得してください
~~~
学務情報サービス＞履修・成績情報＞履修時間割表＞登録する授業名をクリック＞「LMS」欄のジャンプボタンを右クリックし、「リンクをコピー」をクリックし取得する
~~~
3. 「保存する」をクリックし、データを保存。
4. アカンサスポータルを開くと、右下に「LMSを開く」が表示されているので、これをクリック
5. タイムテーブルに先ほど入力した内容が記載されているのを確認し、授業をクリックするとページに飛びます
### 時間割の変更
拡張機能の設定画面（extention options）から授業名とURLを変更し、保存してください。
**まだベータ版のため、１つの教科を入力するごとに保存しておくことをお勧めします。**

## 技術スタック

- 言語: HTML / CSS / Vanilla JavaScript（フレームワークなし）                                                         
- プラットフォーム: Firefox ブラウザ拡張機能（Manifest V3）
- データ保存: chrome.storage.local（ブラウザ組み込みストレージ API）                                                  
                                                                                                                        
ファイル構成と役割                                                                                                    
- manifest.json : 拡張機能の設定・権限定義
- background/background.js : タブを開く処理(バックグラウンド常駐)                                                          
- script/script.js : 大学LMSページにボタンを注入するコンテンツスクリプト                               
- options/options.html+js : 時間割入力・保存の設定画面                                
- index/index.html+js : 時間割表示・授業リンクへの遷移画面                             

## 情報の取り扱い
- 保存場所: ユーザーのPC内（ブラウザのローカルストレージ）のみ
- 保存内容: 講義名とURL（月〜金・1〜7限分）
- 外部送信: 一切なし（サーバーへの通信は行わない）                                                                    
- 取得権限: storage（データ保存）、tabs（タブを開く）の2つのみ                                                        
- アクセス対象ページ: *.acanthus.cis.kanazawa-u.ac.jp のみにスクリプトを注入
