プライバシーポリシー

本プライバシーポリシーは、「Lms Expand Tools」拡張機能（Chrome 版・Firefox 版、以下「本拡張機能」）における個人情報・データの取り扱いについて説明します。

1. 収集するデータ

本拡張機能が収集・保存するデータは以下のみです。

  • ユーザーがオプション画面で入力した時間割データ（授業名・URL）

これ以外の個人情報（氏名・メールアドレス・閲覧履歴など）は一切収集しません。

2. データの保存場所

入力された時間割データは、ブラウザが提供するローカルストレージ API を使用してユーザー自身の端末内にのみ保存されます。データは外部サーバーやクラウドへ送信・アップロードされることはありません。

  • Chromechrome.storage.local を使用します。
  • Firefoxbrowser.storage.local を使用します。

3. データの利用目的

保存したデータは、以下の目的にのみ使用します。

  • 時間割ページ（拡張機能内）への表示
  • オプション画面での設定内容の復元

4. 第三者へのデータ提供

本拡張機能は、いかなる第三者にもユーザーデータを提供・販売・共有しません。

5. 使用する権限とその理由

Chrome 版・Firefox 版ともに以下の権限を使用します。

  • storage：時間割データを端末内に保存・復元するために使用します。
  • tabs：拡張機能内の時間割ページを新しいタブで開くために使用します。
  • ホスト権限（acanthus.cis.kanazawa-u.ac.jp）：金沢大学LMSのページ上にショートカットボタンを表示するために使用します。

バックグラウンド処理の違い

  • ChromeService Worker を使用してバックグラウンド処理を行います。
  • Firefoxバックグラウンドスクリプトを使用してバックグラウンド処理を行います。

6. リモートコード

本拡張機能は外部からコードを取得・実行するリモートコードを一切使用しません。すべてのスクリプトは拡張機能パッケージ内に含まれています。

7. お問い合わせ

本プライバシーポリシーに関するご質問は、GitHub のリポジトリ Issues よりお問い合わせください。

Privacy Policy

This Privacy Policy describes how the "Lms Expand Tools" extension (available for both Chrome and Firefox, the "Extension") handles user data.

1. Data We Collect

The Extension collects and stores only the following data:

  • Timetable data entered by the user in the options page (class names and URLs)

No other personal information (such as names, email addresses, or browsing history) is collected.

2. Where Data Is Stored

All timetable data is stored locally on the user's device using the browser's local storage API. Data is never transmitted to or uploaded to any external server or cloud service.

  • ChromeUses chrome.storage.local.
  • FirefoxUses browser.storage.local.

3. How Data Is Used

Stored data is used solely for the following purposes:

  • Displaying the timetable in the Extension's timetable page
  • Restoring settings in the options page

4. Data Sharing

The Extension does not share, sell, or transfer any user data to third parties under any circumstances.

5. Permissions and Their Justification

Both the Chrome and Firefox versions use the following permissions:

  • storage: Used to save and restore timetable data on the user's device.
  • tabs: Used to open the Extension's timetable page in a new tab.
  • Host permission (acanthus.cis.kanazawa-u.ac.jp): Used to inject a shortcut button into Kanazawa University's LMS pages.

Background Processing Differences

  • ChromeUses a Service Worker for background processing.
  • FirefoxUses a background script for background processing.

6. Remote Code

The Extension does not use any remotely hosted code. All scripts are bundled within the extension package.

7. Contact

For questions regarding this Privacy Policy, please open an issue in the GitHub repository.
